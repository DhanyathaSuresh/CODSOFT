# 🤖 Dino AI - Rule-Based Chatbot

This project was developed as part of my **Artificial Intelligence Internship at CodSoft**.

Dino AI is a **Rule-Based AI Chatbot** built using **Python Flask, HTML, CSS, and JavaScript**. It provides intelligent responses based on predefined rules through a modern, interactive web interface.

---

## 🎯 Objective

To develop a responsive Rule-Based Chatbot that can interact with users and answer predefined questions using Artificial Intelligence concepts.

## 🚀 How to Run

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```
http://127.0.0.1:5000
```
