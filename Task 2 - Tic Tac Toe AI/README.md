# Task 2 - Tic Tac Toe AI Website 🎮🧠

This project is part of my **Artificial Intelligence Internship at CodSoft**.

In this task, I developed a **website-based Tic Tac Toe game** where a human player competes against an intelligent AI opponent. The game runs directly in the browser and allows the user to play against the computer through an interactive interface.

The AI uses the **Minimax Algorithm with Alpha-Beta Pruning** to choose the best possible move. This makes the AI unbeatable, and the best possible result for the human player is usually a draw.

---

## 🎯 Objective

To implement the classic **Tic Tac Toe** game as an interactive website with an unbeatable AI using the **Minimax Algorithm**.

The AI analyzes all possible future game states and selects the most optimal move for every situation.

## 🔄 Game Flow

1. The game opens in the browser.
2. The human player uses `X`.
3. The AI player uses `O`.
4. The human clicks an empty square to make a move.
5. The AI evaluates the board using the Minimax algorithm.
6. The board updates automatically after every move.
7. The game ends when a player wins or the board is full.

## 🧠 Algorithm Used

The AI uses the **Minimax Algorithm with Alpha-Beta Pruning**.

The algorithm checks every possible future move and scores the final result.

| Result | Score |
|--------|------:|
| AI Wins | +10 |
| Human Wins | -10 |
| Draw | 0 |

The AI chooses the move with the highest score, making it unbeatable.  
Alpha-Beta Pruning improves the process by skipping unnecessary calculations.
